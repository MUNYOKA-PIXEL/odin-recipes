# Odin Recipes

## Project Description
This is a basic recipe website built as part of **The Odin Project** curriculum. The goal of this project is to practice and demonstrate foundational HTML skills before moving on to CSS styling.

## Skills Demonstrated
* Initializing a Git repository and managing version control.
* Structuring an HTML document with proper tags (`<html>`, `<head>`, `<body>`).
* Creating headings, paragraphs, and lists.
* Implementing internal and external hyperlinks.
* Organizing project files and directories.
